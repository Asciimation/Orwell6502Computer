These folders contain BASIC program listings entered from the series of 80s Usborne Computer books.

The code is written to run on the Orwell 6502 machine but it should be easy to port it to any 
other machine of the era running Basic. Orwell runs a modified version of the original MS Basic.

More details about Orwell and all the book reviews can be found here: 

http://www.asciimation.co.nz/bb/category/6502-computer
http://www.asciimation.co.nz/bb/category/usborne
